## 📌 About The Project

**Humourastic** is a fully customised **Linktree-style website** built and delivered for a real client. Instead of using a generic Linktree page, the client wanted a **personalised, branded experience** to showcase their curated Spotify playlists and contact information — all in one stylish, animated page.

> 💼 **Type:** Freelance Client Project
> 🚀 **Status:** Live & Deployed
> 🎯 **Purpose:** Custom link-in-bio page with Spotify playlist management
